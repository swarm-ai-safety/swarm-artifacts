\begin{abstract}
We benchmark SWARM coordination mechanisms on a verifiable reasoning track, comparing divergence, critique, reconciliation, and memory retrieval. We report accuracy, disagreement rates, and costs across 500 tasks.
\end{abstract}

\section{Introduction}
We evaluate SWARM-style coordination mechanisms on Track A (verifiable reasoning),
using controlled arithmetic and word-problem tasks with deterministic checks.
Each condition corresponds to a coordination policy (divergence, critique,
reconciliation, memory). This paper summarizes one full run (ID: track\_a\_20260210\_034234).

\section{Methods}
Tasks: 500 total, generated with fixed random seed and difficulty calibration.

\subsection{Key Findings}

\begin{enumerate}
\item \textbf{Voting is essential for adversary resistance.} Without voting, adversary success rate is $\sim$95\%. With voting, it drops to $\sim$24\%.
\item \textbf{Binary tasks are vulnerable.} knights\_knaves achieves 0\% accuracy under adv\_majority (3 adversaries vs 2 honest), as adversaries can outvote the correct binary choice.
\item \textbf{Symbolic reasoning is most robust.} Achieves 95\% accuracy even under sycophant attacks.
\item \textbf{Code verification is weakest.} Consistently 58--61\% across adversarial conditions.
\end{enumerate}

\section{Limitations}
\begin{itemize}
\item Heuristic solvers used (0 tokens); LLM evaluation pending.
\item Confidence treated as reported scalar without calibration.
\item Simple divergence heuristics; richer critics needed.
\item Binary-choice tasks need guaranteed honest majority.
\end{itemize}

\section{Conclusion}
Voting-based coordination provides strong adversary resistance (76.5\% accuracy vs 100\% baseline), reducing adversary success from 95\% to 24\%. Task families vary significantly in robustness: symbolic reasoning resists attacks well (89\%), while code verification (60\%) and binary logic puzzles under majority attack (0\%) remain vulnerable.
